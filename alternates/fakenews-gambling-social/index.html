⚙️ Setup Instructions

1. Pi-hole (DNS Resolver)

  1. Click the copy button next to domains.txt above to copy the absolute link.
  2. Go to your Pi-hole admin console.
  3. Navigate to Group Management -> Adlists.
  4. Paste the copied URL under Address and click Add.

2. AdGuard Home

  1. Click the copy button next to hosts.txt above.
  2. Open AdGuard Home dashboard and navigate to Filters -> DNS Blocklists.
  3. Click Add Blocklist -> Add a custom list.
  4. Paste the copied URL under list URL, give it a name, and save.

3. Browser Adblockers (uBlock Origin / AdGuard Extension)

  1. Copy the link to adblock.txt above.
  2. Open the extension settings and go to the Filter Lists (or My Filters) tab.
  3. Scroll down, check Import under custom lists, paste the URL, and click Apply Changes.
