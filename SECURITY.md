# Security Policy

## Supported Versions

We actively maintain and support the latest versions of compiled lists and automation workflows. 

| Version | Supported |
| ------- | --------- |
| Active (Main Branch) | :white_check_mark: Yes |
| Stale/Archived Branches | :x: No |

## Reporting a Vulnerability

We take the security of this project and the accuracy of our domain intelligence seriously. If you find a security vulnerability, an bypass vector in list compiler scripts, or a major issue that impacts users:

1. **Please do not report security vulnerabilities in public issues.**
2. Send an email directly to the maintainer at:
   📧 **devmasud@proton.me**
3. Include detailed steps to reproduce the issue, along with any relevant proof-of-concept material.

We will review your submission and respond within 48 hours to coordinate a fix. Thank you for helping keep this project secure!
